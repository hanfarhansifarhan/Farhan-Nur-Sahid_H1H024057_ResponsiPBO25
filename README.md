Penjelasan Singkat Code dan Aplikasi 
Deskripsi Aplikasi:
PokéCare adalah sistem simulasi pelatihan Pokémon berbasis web yang memungkinkan pengguna:
1. Melihat informasi dasar Pokémon
2. Melakukan berbagai jenis latihan
3. Mencatat riwayat perkembangan Pokémon
4. Mengamati pengaruh tipe Pokémon terhadap gaya latihan
Penerapan 4 Pilar OOP :
1. Encapsulation - Properti private dengan getter/setter
2. Inheritance - Pewarisan class Pokemon ke BugPoisonPokemon
3. Polymorphism - Override method specialMove() dan train()
4. Abstraction - Abstract class Pokemon dengan abstract methods

Cara Menjalankan Aplikasi PokéCare :
1. Lihat Informasi Pokémon di halaman beranda
2. Klik "Mulai Latihan" untuk training
3. Pilih jenis latihan (Attack/Defense/Speed)
4. Atur intensitas (1-10)
5. Lihat hasil peningkatan level dan HP
6. Cek "Riwayat Latihan" untuk melihat history
